# Steps to Install Browser Extension

Step 1: Clone this repo

Step 2: Open Chrome Extensions page
- In Chrome, go to chrome://extensions in the address bar
- Or: Chrome menu (⋮) → More Tools → Extensions

Step 3: Enable Developer Mode
- Toggle the Developer mode switch in the top-right corner of the Extensions page

Step 4: Load the extension
- Click Load unpacked
- Navigate to and select the unzipped repo
- Click Select Folder

Step 5: Confirm it's working
- You should see the extension appear in your Extensions list
- Pin it to your toolbar by clicking the puzzle piece icon (🧩) → pin the extension

You're ready! Open any paper on Semantic Reader and the extension will activate automatically.
